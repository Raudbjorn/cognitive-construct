Quick Start

1
Clone the repository

Get the cognitive-construct skills package.

git clone https://github.com/Raudbjorn/cognitive-construct.git
cd cognitive-construct
2
Run the setup wizard

Configure API keys with the interactive TUI.

uv run scripts/setup.py
3
Test connectivity

Verify your API keys are working.

uv run scripts/setup.py --test
